Welcome to Jeopardy!

This classic trivia game is for 4 players (one host, three contestants). This console view is for the host only. They are allowed to pick the categories and view the answers.
The game will generate a new tab for both the host sheet and the clue screen for players. The host is responsible for adding the scores to the players via the clue screen.
